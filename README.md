# toDoList as homework 
Creating simple toDoList with adding, deleteing and searching tasks using OOP knowledge


# Criteria:
an app allows user to create a new list by calling from new
adding tasks to the list by entering a task in the text field 
remove the task from the list by clicking the delete button
marking tasks as done by clicking on the task 
using the render method invoked at each state change
searching tasks through an additional text field 
creating searching buttons ALL, COMPLETED, NOT COMPLETED
saving tasks in localStorage
